Detailed Description

The DiskScheduler class acts as a husk class for different disk scheduler implementations.

It provides a basic read/write disk scheduling framework using posix mutex and condition variables to achieve interprocess synchronization. This is not a standalone class and requires extension to achieve full functionality. The insert() and remove() methods must be implemented in order to establish scheduling.

The abstract method deQ must be implemented in order to establish a scheduling order.
